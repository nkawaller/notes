
# Notes - static site generator


## Development

  To start up the tailwind listener, run this script:
  ```
  ./runtailwind.sh
  ```

  To recompile the app, run:
  ```
  go run compileServer.go
  ```

  And then start the server by running:
  ```
  ./bin/runserver
  ```

  Run the tests with:
  ```
  go test ./...
  ```
